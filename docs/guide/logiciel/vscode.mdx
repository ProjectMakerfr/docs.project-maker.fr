# Visual Studio Code (vscode)

## Installation de Visual Studio Code

1. Rendez-vous sur le site officiel pour télécharger VSCode : [Télécharger VSCode](https://code.visualstudio.com/download)
2. Assurez-vous que votre système répond aux exigences minimales : [Configuration requise](https://code.visualstudio.com/docs/supporting/requirements)
3. Consultez la documentation officielle pour en savoir plus sur VSCode : [Documentation de VSCode](https://code.visualstudio.com/docs/)

**Atouts de Visual Studio Code :**
- Gratuit
- Facile à prendre en main
- Personnalisable
- Prise en charge de plugins communautaires et officiels
- Mises à jour régulières
